## Data
- `id` - Unique Identifier
- `title` - Title of the post
- `url` - URL to the post
- `num_points` - Points for the post
- `num_comments` - Comments on the post
- `author` - Author of the post
- `	created_at` - Date posted


## Objectives

Analysis Ask HN and Show HN posts to understand trends about comments and points for each type of post.

## Libraries Used:

- *csv*
- *datetime*
