## Objectives

A mobile service provider is sunsetting a legacy plan. It wants to predict the best fit out of two new plans being launched to recommend to legacy customers by utilising usage data

## Data

The data contains user behaviour information: 

- `сalls` — number of calls,
- `minutes` — total call duration in minutes,
- `messages` — number of text messages,
- `mb_used` — Internet traffic used in MB,
- `is_ultra` — plan for the current month (Ultra - 1, Smart - 0).

## Libraries Used

- _pandas_
- _numpy_
- _scikit-learn_

## Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
