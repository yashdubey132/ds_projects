## Objectives
A mining company is attempting to utilise a model to predict the best region to create new oil wells. The objective is to minimize risk and deliver highest average profits.

## Data
3 Datasets with geopgraphical information about regions. The data is synthetic. The features are not named but are considered significant for our analysis.

The dataset contains the following fields:

- `id` - unique well identifier.
- `f0,f1,f2` - coded characteristics that act as features.
- `product` - volume of reserves (in 1000s of barrels) 

## Libraries Used

- _pandas_
- _numpy_
- _scikit-learn_
- _matplotlib_

## Models Used

- Linear Regression
